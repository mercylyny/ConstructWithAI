PREPROCESSING PIPELINE:
----------------------
The implementation includes a 7-step pipeline optimized for architectural drawings:

Step 1: Grayscale Conversion
   - Converts color images to grayscale
   - Reduces computational complexity
   - Eliminates color noise

Step 2: CLAHE (Contrast Limited Adaptive Histogram Equalization)
   - Parameters: clipLimit=2.0, tileGridSize=(8,8)
   - Enhances local contrast
   - Critical for low-contrast scanned plans
   - Prevents over-amplification of noise

Step 3: Bilateral Filtering (Noise Reduction)
   - Parameters: d=9, sigmaColor=75, sigmaSpace=75
   - Reduces noise while preserving edges
   - Essential for maintaining thin dimension lines
   - Better than Gaussian blur for technical drawings

Step 4: Adaptive Thresholding
   - Method: ADAPTIVE_THRESH_GAUSSIAN_C
   - Parameters: blockSize=11, C=2
   - Converts to binary (black/white) image
   - Handles varying lighting conditions
   - Superior to global thresholding for scanned documents

Step 5: Morphological Closing
   - Kernel: 2x2, iterations=1
   - Connects broken characters
   - Fills small gaps in text
   - Common issue in scanned/photographed plans

Step 6: Resolution Upscaling
   - Minimum dimension: 1000 pixels
   - Method: INTER_CUBIC interpolation
   - Ensures OCR has sufficient detail
   - Prevents accuracy loss on small images

Step 7: Debug Mode (Optional)
   - Saves intermediate steps when debug=True
   - Output directory: outputs/ocr_debug/
   - Files: _1_gray, _2_enhanced, _3_denoised, _4_binary, _5_final

TESSERACT OPTIMIZATION:
----------------------
Configuration optimized for construction plans:

- OEM 1: LSTM neural network mode (best accuracy)
- PSM 6: Uniform block of text (typical for plan annotations)
- Character Whitelist: "0123456789.mMxX-ABCDEFGHIJKLMNOPQRSTUVWXYZ "
  * Numbers: 0-9
  * Decimal points: .
  * Units: m, M (meters)
  * Dimensions: x, X (e.g., "3.5M X 2.8M")
  * Separators: - (dash)
  * Labels: A-Z (uppercase)
  * Spaces: preserved for multi-word labels
- preserve_interword_spaces=1: Maintains spacing in labels
