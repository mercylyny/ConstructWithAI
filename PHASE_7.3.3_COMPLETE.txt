================================================================================
INTERPRETATION LOGIC
================================================================================

1. Wall Detection:
   - Scan each line of clean text
   - Look for wall identifiers (W1, Wall A, etc.)
   - Skip lines with non-wall keywords (DOOR, WINDOW)

2. Measurement Matching:
   - Find measurements on same line as wall ID
   - Convert to meters using normalize_to_meters()
   - Validate realistic range (0.5m - 20m)

3. Duplicate Prevention:
   - Check if wall label already exists
   - Skip duplicate entries

4. Confidence Scoring:
   - High: 3+ walls with clear labels and measurements
   - Medium: 1-2 walls detected
   - Low: No clear walls detected
